We will containerize your application using Docker. This ensures isolation from the existing `openclaw` installation and allows for easy updates.

*   **Frontend (React/Vite)**: Dockerized with Nginx to serve static files. Mapped to EC2 port **3002**.
*   **Backend (Node/Express)**: Dockerized Node.js environment. Mapped to EC2 port **3003**.
*   **Database (PostgreSQL)**: Dockerized Postgres database. Internal only (secure).
*   **CI/CD**: GitHub Actions will automatically deploy changes via SSH when you push to `main`.
